# JaguarLearning

#### 介绍
dart框架Jaguar学习仓库

#### 软件架构
Jaguar https://github.com/Jaguar-dart, 是dart的一款服务端框架，可用来开发dart的服务器端及接口。


#### 主要学习内容
 jaguar: ^2.4.19 
 
 jaguar_orm增删改查
 
 jaguar_query_postgres: 使用PostgresSQL数据库
 
 jaguar_jwt: 用户验证
 
 jaguar_session_jwt: session
 
 jaguar_reflect: 反射
 
 jaguar_serializer: json
 
 jaguar_cache: 缓存
 
 jaguar_orm_gen: 自动生成Bean
 
 
 jaguar_hotreload: 热重载
